# Contributor Covenant Code of Conduct

我們承諾為所有參與者提供無騷擾、尊重且具建設性的協作環境，不因年齡、身體特徵、身心狀況、族裔、性別認同、經驗程度、國籍、外貌、種族、宗教或性傾向而有所差別。

可接受行為包括尊重不同觀點、提供具體回饋、承認錯誤並以社群整體利益為優先。不可接受行為包括人身攻擊、騷擾、公開他人私人資訊或其他不專業行為。

維護者有責任釐清並執行本準則，可刪除不當內容或限制違規者參與。事件回報方式應由倉庫維護者在發布時補入私人聯絡管道。

本準則採用 Contributor Covenant 2.1 的精神，完整版本見 <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>。
